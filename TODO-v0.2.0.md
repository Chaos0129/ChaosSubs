# ChaosSubs v0.2.0 TODO — AI 配音

## 核心功能
- [ ] GPT-SoVITS 声纹克隆 + 中文 TTS
- [ ] Demucs 人声/背景音分离
- [ ] 参考音频自动选择（VAD + 语音密度）
- [ ] 分段生成（每 10-15 句一段）解决长文本退化
- [ ] 混音：中文配音 + 原始背景音
- [ ] 多人声纹支持（pyannote 说话人分离）

## 已验证的发现
- 参考音频必须人声密集、背景干净、prompt_text 精确匹配
- 用原始音频做参考（不用 Demucs 分离的）
- MPS 不可用，只能 CPU
- loudnorm 会改采样率，不能用于 TTS 后处理
- GPT-SoVITS v2 + cut5 分割模式效果最好
- 逐句生成拼接效果差，分段/一次性生成更自然

## 已验证的好参考音频

- 文本: 前職も小さい広告代理店で営業とジムと雑用と掃除と
- 选择标准: 人声连续密集、无背景噪音
